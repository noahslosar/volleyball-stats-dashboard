# Volleyball Player Statistics Dashboard

An interactive Streamlit dashboard for exploring volleyball player statistics with multiple visualizations and filtering options.

## Data Source

The data came from the 2026 Men's Volleyball team at CLU.

## Deployment

This app is deployed on Streamlit Cloud and can be accessed at: clumvbstats2026.streamlit.app

## Technologies Used

- **Streamlit**: Web app framework
- **Plotly**: Interactive visualizations
- **Pandas**: Data manipulation
- **SQLite**: Database
- **Git**: Version control

## Explanation

Basically I took my data from project one which was acquired via webscraping the VolleyMetrics website, to 
then look through data and create some meaningful visuals of volleyball statistics that I actually find interesting.
After working through the issues of learning streamlit, I am pretty happy with the dashboard that I created and overall I
think it does exactly what it needs to, communicate the data that was found.
